Summary¶

MCMC stands for Markov Chain Monte Carlo. The goal of MCMC is to estimate an unknown distribution (posterior distribution) using Bayes' Theorem updating prior probability distribution using new data (likelihood distribution).

  • We set unknown distribution theta to 0.30, this is the actual probability MCMC tries to estimate.
  • We use Bernoulli distribution to produce 10-coin toss (binary 0 or 1).
  • Prior distribution is a beta distribution of alpha=2, beta=2.
  • The likelihood function is a Bernoulli distribution.
  • We use Metropolis Hastings algorithm to run 1000 times MCMC random walk to estimate the target distribution theta. Metropolis Hastings algorithm is an iterative algorithm that rejects (stay to the same state) or accepts (move to a new state) proposals based on the probability of new state is greater than a random number.
  • The algorithm estimates actual theta mean is 0.289, and SD s 0.120.
